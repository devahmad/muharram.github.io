<h2 id="english">🇬🇧 English</h2>

A highly interactive, visually stunning, and multi-lingual web application dedicated to the events of Karbala. Built with modern web technologies, this platform serves as a comprehensive encyclopedia of the personas of Ashura, along with interactive Ziyarat readings featuring synchronized audio.

### ✨ Features
- **Interactive Ziyarat:** Full Arabic text of Ziyarat Ashura & Warith with synchronized translations.
- **Audio Integration:** Listen to the beautiful recitations while reading.
- **Comprehensive Persona Encyclopedia:** An unprecedented database of 274+ historical figures (companions, family members, and adversaries) present at Karbala.
- **Modern UI/UX:** Dark theme, fluid animations powered by `framer-motion`, and fully responsive design.
- **Multi-lingual Support:** Seamlessly switch between English, Persian (فارسی), and Arabic (العربية) with auto RTL/LTR layout adjustments.

### 🚀 Getting Started
1. **Clone the repository:** `git clone https://github.com/devahmad/muharram.github.io.git`
2. **Install dependencies:** `npm install`
3. **Run the server:** `npm run dev`
4. Open `http://localhost:3000`

## 🤝 Contributing
To add a new persona, simply create a `.json` file for the persona in each of the language folders (`src/locales/{fa,en,ar}/personas/`), and export it in the respective `index.ts` files!
